> A weather app using OpenWeatherMap API to displays temperature, humidity and sky conditions written in vanilla js

### 🏠 [Homepage](https://ryder007.github.io/weather-app/)

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
